# NEXUM — site vitrine

Site React/Vite prêt à publier sur Vercel.

## Lancer en local
npm install
npm run dev

## Publier sur Vercel
1. Créer un compte GitHub.
2. Mettre ce dossier dans un nouveau dépôt GitHub.
3. Aller sur vercel.com > Add New Project.
4. Importer le dépôt GitHub.
5. Cliquer sur Deploy.

## Important
Avant de recevoir de l'argent réel ou de proposer une prise de participation, il faut valider le cadre juridique avec un avocat spécialisé/AMF.
